Aoi (青い) style guide

Aoi (Japanese: Blue, Green) is a design language for Emeraldion UX.

This style guide contains a catalog of patterns, as well as guidelines for their use.

Colors

Aoi uses a palette of four tones and eight hues: a main hue called accent, then four hues associated with levels, and three extra hues. Additionally, the theme uses four tones of grayscale. All colors have been tested for color contrast according to WAI WCAG 2.0 AAA guidelines.
